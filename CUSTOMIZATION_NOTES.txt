# FoodNova Customization Guide

## 1. LOGO CUSTOMIZATION

### Navbar Logo (Main Header)
File: `/frontend/src/components/Navbar.jsx` (Lines 24-31)

Current code uses a styled div with letter "F":
```jsx
<div className="w-10 h-10 bg-primary rounded-lg flex items-center justify-center">
  <span className="text-white font-bold text-xl">F</span>
</div>
```

To use your own logo image:
1. Place your logo file in `/frontend/public/logo.png`
2. Replace the code with:
```jsx
<img 
  src="/logo.png"
  alt="FoodNova" 
  className="h-10 w-auto"
/>
```

### Footer Logo
File: `/frontend/src/pages/Home.jsx` (Footer section around line 217)
The footer has text-only branding "FoodNova". Update the text or add an image similarly.

## 3. DELIVERY OPTIONS

The app supports two delivery types:

### Pickup (Free)
- Customer selects "Pickup" at checkout
- No delivery fee
- Shows store pickup location
- Order address marked as "PICKUP - ..."

### Delivery (Fee paid on delivery)
- Customer selects "Delivery" at checkout
- Enters delivery address
- Delivery fee is calculated based on location
- Customer pays delivery fee upon receiving order
- Orange notice displayed: "Delivery fee will be calculated based on your location and paid upon delivery"

## 4. SMS NOTIFICATIONS (Africa's Talking)

SMS notifications are sent for:
- Order placed
- Payment receipt approved
- Payment receipt rejected
- Order status: Paid
- Order status: Confirmed
- Order status: Out for Delivery

### Configuration:
File: `/backend/.env`

```
AT_USERNAME=sandbox
AT_API_KEY=your_africastalking_api_key
```

For production, change `AT_USERNAME` to your Africa's Talking username.

### SMS Templates:
File: `/backend/app/services/sms.py`

You can customize the SMS message templates in this file.

## 6. ORDER STATUS FLOW

Available order statuses:
1. **pending** - Order placed, awaiting payment
2. **paid** - Payment verified
3. **confirmed** - Order confirmed and being prepared
4. **out_for_delivery** - Order dispatched (SMS sent)
5. **cancelled** - Order cancelled

## 7. CURRENCY

All prices display in Nigerian Naira (₦)
Format function in: `/frontend/src/api/store.js`

```js
export const formatPrice = (amount) => {
  return `₦${amount.toLocaleString()}`;
};
```

To change currency symbol, update this function.

## 10. DEPLOYMENT

### Backend (Render)
1. Set root directory: `backend`
2. Build command: `pip install -r requirements.txt`
3. Start command: `uvicorn app.main:app --host 0.0.0.0 --port 10000`

Environment variables required:
- DATABASE_URL (PostgreSQL)
- JWT_SECRET
- ADMIN_EMAIL
- ADMIN_PASSWORD
- FRONTEND_ORIGIN (Vercel URL)
- AT_USERNAME
- AT_API_KEY

### Frontend (Vercel)
1. Set root directory: `frontend`
2. Build command: `yarn build`
3. Output directory: `build`

Environment variables:
- REACT_APP_BACKEND_URL (Render backend URL)
